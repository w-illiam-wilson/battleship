
## Description

Battleship backend

## Installation
Use node version >= 16

```bash
$ npm install
```

## Setup

Ensure postgres is running, then go into app.module and change the username/password/etc 

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Postman collection

Use the postman collection to interact with the service.
